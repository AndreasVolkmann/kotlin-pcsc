fun connect(reader: String, shareMode: ShareMode, preferredProtcols: Set<Protocol>?): Card

Connects to a given reader.

Equivalent to SCardConnect.

Parameters

reader - Reader name from listReaders

shareMode - Flag to indicate if other applications may form connections to the card

preferredProtcols - A set of protocols to use when communicating. If null, then no protocol negotiation will take place.

Exceptions

PCSCError -

Return
Card object to communicate with the card. The protocol in use is avaliable as Card.protocol
